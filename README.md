# MIST Rulebook (Unofficial)

A searchable, well-organized website version of the MIST 2026 Competition Rulebook and study materials.

**Live Site:** [mist-rulebook.netlify.app](https://mist-rulebook.netlify.app)

## About

This is an **unofficial** markdown version of the MIST (Muslim Interscholastic Tournament) rulebook and competition materials. The goal is to make the rules more accessible, searchable, and easier to navigate than the original PDF documents.

**For official and authoritative rules, always refer to [getmistified.com](https://getmistified.com).**

## Features

- **Full-text search** - Find any rule or topic instantly
- **Mobile-friendly** - Access rules on any device
- **Role-based guides** - Quick start guides for Students, Coaches, and Judges
- **Organized navigation** - Competitions grouped by category
- **What's New in 2026** - All key changes highlighted
- **Quiz Bowl topics** - Study materials for all 5 topics
- **Glossary** - Definitions of key MIST terms
- **PDF downloads** - Original documents available

## Tech Stack

- [Astro](https://astro.build/) with [Starlight](https://starlight.astro.build/) - Documentation framework
- [Pagefind](https://pagefind.app/) - Static search
- [Netlify](https://netlify.com/) - Hosting

## Development

```bash
# Install dependencies
cd website
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
