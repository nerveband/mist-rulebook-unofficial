## Quick Facts

| Competition | Type | Early Submission |
|-------------|------|------------------|
| Extemporaneous Essay | Individual, Written | No |
| Extemporaneous Speaking | Individual, Oral | No |
| Original Oratory | Individual, Oral | No |
| Poetry | Individual, Written | **Yes** - 2 weeks prior |
| Prepared Essay | Individual, Written | **Yes** - 2 weeks prior |
| Short Fiction | Individual, Written | **Yes** - 2 weeks prior |
| Spoken Word | Individual, Performance | No |

:::tip[Early Submission]
Poetry, Prepared Essay, and Short Fiction require submission via MyMIST Portal two weeks before your tournament. See [Early Submissions](/rulebook/early-submissions/) for details.
:::
